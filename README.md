# ManMachine

ManMachine is a game for Unity 1 week game jam takes place in unityroom.

[https://unityroom.com/unity1weeks](https://unityroom.com/unity1weeks)

# Instruction

## Move

Keyboard: AWSD / KJ
Finger: Swiping
Mouse: Dragging

## Switch Camera

Keyboard: UI
Other: Buttons on HUD

## Interact

Keyboard: Space
Finger: Tap
Mouse: Click
